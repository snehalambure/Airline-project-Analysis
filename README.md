# ✈️ Airline Project Analysis  

This project analyzes airline data using **SQL, Power BI, Tableau, and Excel** to find insights such as passenger load factor, top carriers, routes, and distance group analysis.  

---

## 📂 Repository Contents  
- **SQL Queries** → `My SQL Project.sql`  
- **Power BI Dashboard** → `Power BI Project.pbix`  
- **Tableau Dashboard** → `Tableau Project.twbx`  
- **Excel Data** → `AirlineData.xlsx`  
- **PDF Report** → `Report.pdf`  

---

## 🛠️ Tools & Technologies  
- **SQL** – Data extraction & queries  
- **Power BI** – Data visualization  
- **Tableau** – Dashboard design  
- **Excel** – Data preparation  

---

## 🚀 Key Insights  
- Yearly, Quarterly & Monthly Load Factor %  
- Carrier performance (Top 10 airlines by passenger preference)  
- Weekend vs Weekday passenger trends  
- Distance group analysis (short, medium, long, ultra-long haul)  
- Top routes by number of flights
